Error code of un unsuccessful trade

属性值

ETradeErrorCode

示例

var mySymbol = MarketData.GetSymbol("EURUSD");
TradeResult result = ExecuteMarketOrder(ETradeDirection.Sell, mySymbol, 1);

    Print("Error: {0}", result.Error);
